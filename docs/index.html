📧 Communication Service

Multi-channel Communication Platform for Notifications & Webhooks

v1.0.0

🎯 Overview

The Communication Service handles all platform notifications including email, SMS, push notifications, and webhook delivery with real-time updates.

🛠️ Tech Stack

  • Node.js 18+ / NestJS 11
  • PostgreSQL + TypeORM
  • Redis + Kafka
  • Socket.IO for real-time
  • Handlebars templates

🚀 Quick Start

# Install dependencies
npm install

# Configure environment
cp env.development .env

# Start server
npm run start:dev

⚙️ Environment Variables

Variable Description
PORT HTTP server port (5006)
GRPC_PORT gRPC server port (50006)
AWS_ACCESS_KEY_ID AWS SES access key
AWS_SECRET_ACCESS_KEY AWS SES secret
TWILIO_ACCOUNT_SID Twilio account SID
TWILIO_AUTH_TOKEN Twilio auth token
FIREBASE_PROJECT_ID Firebase project ID

🔌 Communication Channels

📧 Email (AWS SES)

  • Transactional emails
  • Password reset
  • Verification emails
  • HTML templates
  • Delivery tracking

📱 SMS (Twilio)

  • Transaction alerts
  • OTP messages
  • Marketing (opt-in)
  • Delivery receipts

🔔 Push (Firebase)

  • Real-time alerts
  • System notifications
  • iOS & Android
  • Topic messaging

🔗 Webhooks

  • Inbound processing
  • Outbound delivery
  • Automatic retries
  • Signature verification

⚡ Rate Limiting

Channel Default Limit
Email 100 / minute
SMS 50 / minute
Push 1000 / minute
